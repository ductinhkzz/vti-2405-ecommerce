# This is a e-commerce project

## medusa-store

- Create .env
- Copy/pass and change environment variable in .env.template
- Run source code

```(bash)
  npm i
  npm run seed
  npm run dev
```

Note: "npm run seed" just run 1 time when setup this repo. The next time, just run "npm run dev"
